# 📊 Mobile Sales Dashboard (Power BI)

## 📌 Overview
This project showcases an interactive Power BI dashboard designed to analyze mobile sales performance across different dimensions such as city, brand, payment method, and time.
The dashboard helps in identifying key trends, customer behavior, and business insights for better decision-making.\

<img width="2400" height="1347" alt="image" src="https://github.com/user-attachments/assets/573fc288-bf19-4055-b355-c62f68477635" />


## 🚀 Key Highlights
- Total Sales: 769M
- Total Quantity Sold: 19K
- Total Transactions: 4K
- Average Price: 40K
---
## 📊 Dashboard Features
- 📍 City-wise Sales Analysis (Map View)
- 📅 Monthly Filtering (Jan–Dec)
- 📱 Mobile Model Performance
- 💳 Payment Method Distribution
- ⭐ Customer Ratings Analysis
- 📈 Daily Sales & Quantity Trends
- 🏷️ Brand-wise Performance Table
---
## 🛠️ Tools & Technologies Used
- Power BI
- Data Visualization
- Data Cleaning
- DAX (Data Analysis Expressions)
